## Repository Purpose

This repository contains the source code for my public static CV.  
The goal is to keep a fast, accessible, SEO-friendly, and easy-to-print version of my professional profile available online.

It is intentionally simple: a single static HTML page, optimized for clarity, ATS readability, printing, and fast loading.
